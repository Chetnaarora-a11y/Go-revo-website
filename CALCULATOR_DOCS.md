# Solar Savings Calculator Documentation

This document describes the calculations and visual configurations used in the interactive Solar Savings Calculator.

## Calculations Configuration
- **Tariff rate**: Rs. 7.5 per unit (configurable in `custom.js`)
- **Generation factor**: 120 kWh per month per kW (average generation capacity in India)
- **Interest rate**: 9.5% per annum for loan EMIs
- **Tenure**: 60 months (5 years)

## PM Surya Ghar Subsidy Scheme Rates
- 1 kW: Rs. 30,000
- 2 kW: Rs. 60,000
- 3 kW or higher: Rs. 78,000 (maximum subsidy capped)

## Layout Styling
- **Desktop**: Two-panel design system with inputs on left and outputs on right.
- **Mobile**: Responsive full-screen drawer overlay sliding from bottom, featuring sticky CTA buttons.
